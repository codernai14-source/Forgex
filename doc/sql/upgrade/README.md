# Forgex 数据库升级脚本

> 版本：**V0.6.5**

本目录是交付包 `database-upgrade/` 的源码目录，用于收集已有环境升级时需要人工确认执行的数据库 SQL。

## 执行原则

- 升级前必须备份目标数据库，并确认目标环境、租户、命名空间和当前应用版本。
- 按 SQL 文件名中的日期顺序执行；同一天存在多个脚本时，按交付说明或文件名顺序执行。
- 执行前先阅读脚本头部说明、影响范围和幂等判断；禁止在未确认目标库状态时批量执行。
- 应用升级脚本不会自动执行 SQL，数据库升级由现场运维或实施人员确认后手工执行。
- 如现场环境已人工调整过同名表、字段、索引或数据，请先在预发环境验证脚本。

## 文件说明

本目录只放进入升级包的正式升级 SQL。开发过程中的诊断脚本、临时修复草稿、排查记录和阶段性资料仍统一放在 `doc/` 目录其他位置，不进入 `Forgex_Doc` 文档中心。
