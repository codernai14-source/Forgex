# 图标选择器使用方式

> 分类：前端 / 组件与页面  
> 版本：**V0.6.0**  
> 更新时间：**2026-04-17**

本文档说明在页面和表单中如何接入 `IconPicker`，重点覆盖代码如何使用、页面如何操作以及常见错误写法。

## 代码位置

```text
Forgex_MOM/Forgex_Fronted/src/components/common/IconPicker.vue
Forgex_MOM/Forgex_Fronted/src/views/system/menu/index.vue
```

## 最小接入示例

```vue
<template>
  <a-form-item label="图标" name="icon">
    <IconPicker
      v-model:value="form.icon"
      placeholder="请选择图标"
    />
  </a-form-item>
</template>

<script setup lang="ts">
import { reactive } from 'vue'
import IconPicker from '@/components/common/IconPicker.vue'

const form = reactive({
  icon: '',
})
</script>
```

## 代码如何使用

页面接入时只需要做三件事：

1. 引入 `IconPicker`
2. 用 `v-model:value` 绑定字段
3. 保存时把该字段一起提交给后端

保存后的值示例：

```ts
{
  icon: 'UserOutlined'
}
```

## 页面如何使用

用户在页面中的操作路径如下：

1. 表单中点击右侧图标按钮
2. 打开图标选择弹窗
3. 通过关键字搜索图标
4. 点击图标项完成选择
5. 表单输入框显示图标名称，按钮显示图标预览

如果用户知道准确的图标名，也可以直接在输入框里手动输入。

## 菜单页面中的标准写法

参考：

```text
Forgex_MOM/Forgex_Fronted/src/views/system/menu/index.vue
```

典型接入代码如下：

```vue
<a-form-item :label="t('system.menu.icon')" name="icon">
  <IconPicker
    v-model:value="formData.icon"
    :placeholder="t('system.menu.form.iconPlaceholder')"
  />
</a-form-item>
```

这种写法适合：

- 菜单配置
- 模块配置
- 配置型表单

## 如何在列表中显示已保存图标

表单保存后，列表页通常会拿到 `record.icon` 这样的字符串。  
渲染时不要直接把字符串写在页面上，而是要转成组件。

菜单页参考写法：

```vue
<template #icon="{ record }">
  <component v-if="record.icon" :is="getIcon(record.icon)" />
  <span v-else>-</span>
</template>
```

这说明：

- `IconPicker` 负责“选”
- `getIcon()` 或其他统一工具负责“渲染”

## 表单校验建议

通常图标字段不是强制项。  
如果业务要求必填，可以加普通表单规则：

```ts
icon: [{ required: true, message: '请选择图标', trigger: 'change' }]
```

## 推荐使用方式

- 把图标字段当作普通字符串字段处理
- 编辑回显时直接把后端返回值赋给表单
- 展示时统一通过工具函数解析

## 不推荐写法

不推荐：

```ts
form.icon = Icons.UserOutlined
```

推荐：

```ts
form.icon = 'UserOutlined'
```

原因是后端无法稳定保存组件对象。

## 常见排查

### 保存成功但列表不显示图标

优先检查：

1. 后端返回的是否仍是图标名字符串
2. 列表页是否调用了 `getIcon(record.icon)`

### 编辑弹窗里图标预览为空

优先检查：

1. `v-model:value` 是否绑定到了正确字段
2. 编辑详情接口是否把 `icon` 正确回填到了表单

### 用户手输一个不存在的图标名

处理建议：

- 保存前允许为空但不建议允许无效值
- 页面展示时对未知图标做兜底
