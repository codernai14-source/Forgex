## 适用场景

- 做多语言字段录入、标题与正文配置：优先看多语言输入两篇文档。
- 做首页、工作台、拖拽布局类页面：优先看个人首页与可拖拽布局两篇文档。
- 处理历史页面中已存在的硬编码中文展示或提示：先确认 `src/utils/legacyI18n.ts` 与 `src/locales/legacyText.ts` 是否已有兼容翻译；新增页面不要继续依赖该兼容层，应直接使用 `t(...)` 或后端统一返回消息。
