# 多语言输入与动态布局

> 分类：前端 / 国际化与布局  
> 版本：**预览版 V0.5.0**

这是前端国际化输入与布局能力的镜像入口页。

## 一、多语言输入

`I18nInput` 主要解决：

- 多语言名称/标题/正文编辑
- JSON 结构化存储与回填
- 简单模式 / 表格模式 / 弹窗模式
- 占位符插入与模板场景支持

原文：
- [多语言输入框组件使用指南](../../../doc/开发规范/04-功能模块/07-公共组件/多语言输入框组件使用指南.md)

## 二、动态布局小组件

动态布局小组件主要用于：

- 仪表盘与首页组件组合
- 页面布局模块化管理
- 可配置、可扩展的前端展示能力
