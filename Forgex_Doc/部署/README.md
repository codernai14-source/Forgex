# 部署文档导航

> 版本：**预览版 V0.6.0**  
> 更新时间：**2026-04-16**

本目录用于统一维护 Forgex 的部署、授权、环境配置与交付说明。

## 一、目录内容

| 位置 | 说明 |
|---|---|
| `README.md` | 部署文档导航 |
| `授权说明.md` | 开发、测试、演示、生产环境授权步骤说明 |
| `nacos配置/` | Nacos 配置模板与示例 |
| `数据库初始化脚本/` | 数据库初始化资料 |

## 二、优先阅读

1. 先阅读 `授权说明.md`
2. 再按目标环境选择对应的 Nacos 配置
3. 最后结合 `Forgex_build/README.md` 选择 Windows 或 Linux 交付目录

## 三、关键说明

- 当前 Forgex 授权校验由后端 `forgex.license` 与 `forgex.deployment` 配置统一控制。
- 系统首次启动时，如果授权目录中没有 `request-info.json`，会自动生成该文件。
- 登录页提示“系统未授权，请先导入有效授权”时，通常表示：
  - 授权目录中不存在 `license.lic`
  - `license.lic` 与当前机器码不匹配
  - `license.lic` 与当前客户码不匹配
  - 授权文件已过期或签名无效
