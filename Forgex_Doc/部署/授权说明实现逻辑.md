# 授权说明实现逻辑

> 分类：部署  
> 版本：**V0.6.0**  
> 更新时间：**2026-04-17**

本文档说明 Forgex 授权文件的生成、装载与校验链路。

## 授权主链路

```text
系统启动
  -> 读取 license-dir
  -> 若无 request-info.json 则自动生成
  -> 授权签发端根据 request-info.json 生成 license.lic
  -> license.lic 放回授权目录
  -> 网关校验授权状态
  -> 放行或阻断受保护接口
```

## 关键配置

| 配置 | 作用 |
|---|---|
| `forgex.deployment.instance-code` | 实例编码 |
| `forgex.deployment.license-dir` | 授权目录 |
| `forgex.license.enabled` | 是否启用授权校验 |
| `forgex.license.public-key` | 公钥 |

## 关键文件

| 文件 | 作用 |
|---|---|
| `request-info.json` | 授权申请文件 |
| `license.lic` | 授权文件 |
| `activation-history.json` | 授权历史 |

## 运行期校验

网关会在登录态判断前先校验授权状态，并根据状态返回未授权、过期、宽限期或无效授权提示。
