# 认证授权

> 分类：后端 / 身份与权限  
> 版本：**预览版 V0.5.0**

这是 `04-功能模块/01-认证授权` 的镜像入口页，用于先快速理解能力边界，再跳转到原始长文阅读细节。

## 一、覆盖范围

当前认证授权链路主要覆盖：

- 账号登录、注册、登出、重置密码
- 图片验证码、滑块验证码
- 管理员安全校验
- 租户选择与登录后租户切换
- 权限加载与动态路由基础数据来源
- 第三方登录回调（微信、钉钉）

原文重点包括：

1. 权限模型：模块 / 菜单 / 按钮
2. Auth 与 Common 的职责边界
3. 登录服务结构与控制器入口
4. 权限拦截器与 `@RequirePerm` 链路
5. 数据权限与角色数据范围逻辑
