# 报表中心

> 分类：后端 / 模块专题  
> 版本：**V0.5.0**  
> 更新时间：**2026-04-15**

本专题聚焦 `Forgex_Report` 的报表分类、模板、数据源与前端设计器入口。

## 一、模块定位

`Forgex_Report` 负责报表中心的分类管理、模板管理、数据源管理以及双引擎集成。

当前服务名：`forgex-report`  
默认端口：`8084`

## 二、后端入口

### 2.1 控制器

| 控制器 | 说明 |
|---|---|
| `ReportCategoryController` | 报表分类 |
| `ReportDatasourceController` | 数据源管理与测试 |
| `ReportTemplateController` | 模板管理、内容保存、导入导出 |

### 2.2 当前接口分组

| 能力 | 典型路径前缀 |
|---|---|
| 报表分类 | `/report/category/*` |
| 数据源 | `/report/datasource/*` |
| 模板 | `/api/report/template/*`、`/report/template/*` |

## 三、前端入口

### 3.1 页面目录

| 页面 | 路径 | 说明 |
|---|---|---|
| 报表模板页 | `src/views/report/index.vue` | 模板列表与入口页 |
| 数据源页 | `src/views/report/datasource/index.vue` | 数据源管理 |
| 报表设计器 | `src/views/report/components/ReportDesigner.vue` | 设计器承载页 |
| 模板表单 | `src/views/report/components/ReportForm.vue` | 模板编辑表单 |
| 模板预览 | `src/views/report/components/ReportPreview.vue` | 报表预览 |

### 3.2 API 封装

当前前端统一在 `src/api/report/index.ts` 和 `src/api/report/types.ts` 中维护报表接口，已覆盖：

- 模板分页、按 ID 查询、按编码查询、保存、删除
- 设计器 URL 与预览 URL 生成
- 数据源分页、详情、保存、删除、测试
- 分类分页、分类树、保存、删除

## 四、双引擎支持

当前文档与代码明确支持：

- `UReport2`
- `JimuReport`

前端通过 `getDesignerUrl` / `getPreviewUrl` 生成不同引擎的设计器和预览地址。

## 五、当前能力边界

| 能力 | 当前结论 |
|---|---|
| 报表分类 | 已支持 |
| 数据源管理 | 已支持 |
| 模板管理 | 已支持 |
| 模板导入导出 | 已支持 |
| 设计器集成 | 已支持双引擎 |
| 预览入口 | 已支持 |
