# 导入导出

> 分类：后端 / 模块专题  
> 版本：**V0.5.0**  
> 更新时间：**2026-04-13**

## 一、概述

Forgex 提供 **配置驱动** 的 Excel 导入导出能力。导入导出模板、列映射、样式等全部在数据库配置，业务代码只需调用统一接口即可完成文件生成与数据解析。

核心模块位于 `Forgex_Common`（通用能力）和 `Forgex_Sys`（业务接入）。

## 二、核心组件

| 组件 | 包路径 | 作用 |
|---|---|---|
| `ExcelConfigService` | `com.forgex.common.service.excel` | 导入导出配置 CRUD |
| `ExcelFileService` | `com.forgex.common.service.excel` | 模板生成、文件导出、文件解析 |
| `ExcelConfigController` | `com.forgex.sys.controller` | 配置管理与导入导出接口 |
| `ExcelExportService` | `com.forgex.sys.service` | 业务数据导出服务 |
| `TemplateOptionProviderRegistry` | `com.forgex.common.service.excel.provider` | 下拉选项数据源 Provider 注册中心 |
| `ExcelPromptEnum` | `com.forgex.common.enums` | Excel 模块国际化提示枚举 |

## 三、导出配置

### 3.1 数据表结构

**导出配置主表 `fx_excel_export_config`：**

| 字段 | 说明 |
|---|---|
| `id` | 主键 |
| `table_name` | 关联业务表名 |
| `table_code` | 配置编码（唯一标识） |
| `tenant_id` | 租户 ID |

**导出配置子项表 `fx_excel_export_config_item`：**

| 字段 | 说明 |
|---|---|
| `config_id` | 关联主表 ID |
| `export_field` | 导出字段名 |
| `field_name` | 显示列名 |
| `i18n_json` | 多语言列名 JSON |
| `header_style_json` | 表头样式 JSON |
| `cell_style_json` | 单元格样式 JSON |
| `order_num` | 排序号 |

### 3.2 导出流程

```
1. 前端发起导出请求（携带 tableCode + 查询条件）
2. ExcelExportService 根据 tableCode 获取导出配置
3. 按查询条件加载业务数据
4. ExcelFileService.generateExportFile() 生成 xlsx/csv
5. 返回文件流给前端下载
```

### 3.3 样式配置

样式通过 JSON 存储在数据库中，支持以下属性：

```json
{
  "backgroundColor": "#4472C4",
  "fontSize": 12,
  "fontWeight": "bold",
  "align": "center",
  "wordWrap": true
}
```

## 四、导入配置

### 4.1 数据表结构

**导入配置主表 `fx_excel_import_config`：**

| 字段 | 说明 |
|---|---|
| `id` | 主键 |
| `table_name` | 关联业务表名 |
| `table_code` | 配置编码 |
| `tenant_id` | 租户 ID |

**导入配置子项表 `fx_excel_import_config_item`：**

| 字段 | 说明 |
|---|---|
| `config_id` | 关联主表 ID |
| `import_field` | 导入字段名 |
| `field_name` | Excel 列名 |
| `i18n_json` | 多语言列名 JSON |
| `required` | 是否必填 |
| `data_source_type` | 下拉数据源类型：DICT / JSON / PROVIDER |
| `data_source_dict` | 字典编码（数据源为 DICT 时） |
| `data_source_json` | JSON 数组（数据源为 JSON 时） |
| `data_source_provider_code` | Provider 编码 |
| `data_source_provider_field` | Provider 字段 |
| `order_num` | 排序号 |

### 4.2 导入模板下载

```
POST /sys/excel/template/download
Body: { "tableCode": "xxx" }

流程：
1. 根据 tableCode 获取导入配置
2. ExcelFileService.generateTemplate() 生成模板
3. 模板包含：列名（多语言）、数据校验、下拉选项
4. 返回 xlsx 文件流
```

### 4.3 下拉选项 Provider

支持三种数据源类型：

| 类型 | 说明 | 示例 |
|---|---|---|
| `DICT` | 从数据字典获取 | 字典编码 `sys_status` |
| `JSON` | 静态 JSON 数组 | `[{"label":"是","value":"1"}]` |
| `PROVIDER` | 自定义 Provider | 注册编码如 `userProvider` |

**注册自定义 Provider：**

```java
@Component
public class UserOptionProvider implements TemplateOptionProvider {
    @Override
    public String getCode() { return "userProvider"; }

    @Override
    public List<OptionItem> getOptions(String field) {
        // 返回选项列表
    }
}

// 自动注册到 TemplateOptionProviderRegistry
```

### 4.4 导入解析

```
POST /sys/excel/import
Form: file + tableCode

流程：
1. 根据 tableCode 获取导入配置
2. ExcelFileService.parseImportFile() 解析 Excel
3. 按配置做字段映射与数据校验
4. 返回解析后的数据列表
5. 业务侧自行入库
```

## 六、国际化支持

### 6.1 列名国际化

导入导出配置的列名通过 `i18n_json` 字段支持多语言：

```json
{
  "zh-CN": "用户名",
  "en-US": "Username",
  "zh-TW": "使用者名稱"
}
```

导出/模板生成时根据 `LangContext.get()` 选择对应语言的列名。

### 6.2 提示消息国际化

通过 `ExcelPromptEnum` 管理：

| 枚举值 | 默认模板 |
|---|---|
| `EXCEL_IMPORT_SUCCESS` | Excel 导入成功 |
| `EXCEL_IMPORT_FAILED` | Excel 导入失败 |
| `EXCEL_EXPORT_SUCCESS` | Excel 导出成功 |
| `EXCEL_EXPORT_FAILED` | Excel 导出失败 |
| `EXCEL_TEMPLATE_NOT_FOUND` | Excel 模板不存在 |
| `EXCEL_DATA_VALIDATION_FAILED` | Excel 数据校验失败 |
| `EXCEL_ROW_IMPORT_SUCCESS` | 成功导入{0}行数据 |
| `EXCEL_ROW_IMPORT_FAILED` | 失败{0}行数据 |

## 七、业务接入示例

```java
@Service
public class MyModuleExportService {

    @Resource
    private ExcelConfigService excelConfigService;
    @Resource
    private ExcelFileService excelFileService;

    public void export(HttpServletResponse response, MyQueryDTO query) {
        // 1. 获取导出配置
        FxExcelExportConfigDTO config = excelConfigService.getExportConfigByCode("my_module_table");
        // 2. 查询业务数据
        List<Map<String, Object>> dataRows = queryData(query);
        // 3. 生成 Excel 并写入响应流
        excelFileService.generateExportFile(config, dataRows, response.getOutputStream());
    }
}
```
