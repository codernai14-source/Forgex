# 基础数据与物料

> 分类：后端 / 模块专题  
> 版本：**V0.5.0**  
> 更新时间：**2026-04-15**

本专题用于说明 `Forgex_Basic` 模块当前已落地的基础业务扩展位、物料管理能力和前后端对应入口。

## 一、模块定位

`Forgex_Basic` 是 Forgex 后端中的基础业务扩展模块，当前主要承载：

- 基础业务模块健康检查与启用状态探针
- 物料管理能力
- 后续基础数据类业务的扩展入口

当前服务名：`forgex-basic`  
默认端口：`9003`

## 二、当前代码入口

### 2.1 后端服务入口

- 启动类：`Forgex_Basic/src/main/java/com/forgex/basic/ForgexBasicApplication.java`
- 健康检查：`BasicModuleController`
- 物料管理：`BasicMaterialController`

### 2.2 已确认的后端接口

| 能力 | 路径前缀 | 说明 |
|---|---|---|
| 模块探针 | `/module/ping` | 返回 `moduleCode`、`bizEnabled`、`status` 等基础状态 |
| 物料分页 | `/basic/material/page` | 分页查询物料 |
| 物料详情 | `/basic/material/detail` | 查询物料主数据与扩展信息 |
| 物料新增 | `/basic/material/create` | 新增物料，支持自动编码 |
| 物料更新 | `/basic/material/update` | 更新物料及扩展信息 |
| 物料删除 | `/basic/material/delete` | 删除物料 |

### 2.3 前端对应入口

| 页面/接口 | 路径 | 说明 |
|---|---|---|
| 基础模块首页 | `src/views/basic/dashboard/index.vue` | 基础业务首页或占位页 |
| 编码规则页面 | `src/views/basic/encodeRule/index.vue` | 基础业务下的编码规则使用入口 |
| 物料需求来源 | `doc/需求文档/基础数据模块/*` | 当前仍以历史需求文档为详细补充 |

## 三、物料管理能力边界

根据当前控制器与历史需求文档，物料管理已明确包含：

- 物料主数据维护
- 物料编码自动生成
- 租户隔离
- 审批状态字段
- 扩展字段/扩展模块信息
- 与字典、编码规则、导入导出能力联动

建议将物料管理视为 `Forgex_Basic` 中首个正式业务子模块，后续基础数据能力继续沿用该模块承接。

## 五、注意事项

- `BasicModuleController` 当前返回 `bizEnabled=false`，说明基础业务模块仍处于“骨架已就绪、业务逐步启用”的阶段。
- 物料管理的更详细表结构、字段说明和需求背景仍沉淀在历史资料中，正式入口先以本页为准，详细补充可引用 `doc/需求文档/基础数据模块/物料管理后端需求文档兼开发文档.md`。
- 本专题只描述正式入口和当前落地能力，不替代完整接口手册。
