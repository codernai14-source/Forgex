# 任务调度使用方式

> 分类：后端 / 模块专题  
> 版本：**V0.6.0**  
> 更新时间：**2026-04-17**

本文档说明任务调度模块的使用建议。

## 推荐实践

1. 轻量异步任务优先走现有调度骨架。
2. 任务涉及租户、用户、语言时，复制上下文。
3. 任务结果通知尽量复用消息模板能力。

## 常见排查

| 现象 | 排查点 |
|---|---|
| 任务不执行 | 调度开关、启动类、定时表达式 |
| 任务上下文错乱 | TenantContext / UserContext / LangContext |
| 通知不发送 | 消息模板、SSE、下游服务 |
