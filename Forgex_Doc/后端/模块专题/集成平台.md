# 集成平台

> 分类：后端 / 模块专题  
> 版本：**V0.5.0**  
> 更新时间：**2026-04-15**

本专题用于说明 `Forgex_Integration` 模块及前端 `integrationPlatform` 页面所对应的第三方系统集成能力。

## 一、模块定位

`Forgex_Integration` 是 Forgex 对外系统集成模块，当前主要承载：

- 第三方系统管理
- API 接口配置
- 参数配置与参数映射
- 第三方授权
- 调用记录追踪
- 集成运行与治理入口

当前服务名：`forgex-integration`  
默认端口：`9004`

## 二、后端入口

### 2.1 模块结构

- 启动类：`Forgex_Integration/src/main/java/com/forgex/integration/ForgexIntegrationApplication.java`
- 控制器目录：`com.forgex.integration.controller`

### 2.2 当前控制器

| 控制器 | 说明 |
|---|---|
| `ThirdSystemController` | 第三方系统管理 |
| `ApiConfigController` | API 接口配置 |
| `ApiParamConfigController` | API 参数配置 |
| `ApiParamMappingController` | API 参数映射 |
| `ThirdAuthorizationController` | 第三方授权管理 |
| `ApiCallLogController` | API 调用日志 |
| `IntegrationController` | 模块级综合入口 |

## 三、前端入口

### 3.1 页面目录

| 页面 | 路径 | 说明 |
|---|---|---|
| 第三方系统 | `src/views/integrationPlatform/thirdSystem/index.vue` | 系统档案、授权入口 |
| API 配置 | `src/views/integrationPlatform/apiConfig/index.vue` | 接口、调用方式、超时与重试配置 |
| API 调用日志 | `src/views/integrationPlatform/apiCallLog/index.vue` | 调用记录、耗时、请求响应明细 |

### 3.2 API 封装

当前前端统一在 `src/api/system/integration.ts` 中维护集成平台 API，已覆盖：

- 第三方系统列表、详情、新增、更新、删除、批量删除
- API 配置列表、详情、新增、更新、删除、启停
- API 调用日志分页与详情

## 四、当前能力边界

| 能力域 | 当前结论 |
|---|---|
| 集成对象管理 | 已支持第三方系统档案管理 |
| 接口配置 | 已支持 API 编码、路径、请求方式、调用方式、超时与重试 |
| 参数治理 | 已有参数配置与参数映射控制器 |
| 调用审计 | 已支持调用日志记录与详情查看 |
| 授权管理 | 已有第三方授权控制器，适合作为 OAuth/Token 接入扩展位 |

## 五、推荐阅读路径

1. 先看本页了解模块边界
2. 再看前端 `integrationPlatform/*` 页面定位实际操作入口
3. 需要接入平台级消息、租户、权限时，再看对应公共能力文档
