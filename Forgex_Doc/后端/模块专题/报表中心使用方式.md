# 报表中心使用方式

> 分类：后端 / 模块专题
> 版本：**V0.6.5**
> 更新时间：**2026-04-17**

本文档说明报表中心在开发和联调中的推荐使用方式。

## 常用页面

| 页面 | 用途 |
|---|---|
| `src/views/report/index.vue` | 模板列表 |
| `src/views/report/datasource/index.vue` | 数据源管理 |
| `ReportDesigner.vue` | 设计器承载 |
| `ReportPreview.vue` | 报表预览 |

## 接入建议

1. 先建分类，再建数据源，再建模板。
2. 模板编码保持稳定，避免前后端预览地址失效。
3. 双引擎混用时，明确模板对应引擎类型。

## 常见排查

| 现象 | 排查点 |
|---|---|
| 设计器打不开 | 设计器 URL、引擎配置、权限 |
| 报表无数据 | 数据源连接、SQL、参数绑定 |
| 预览异常 | 模板内容、预览地址、引擎兼容性 |
