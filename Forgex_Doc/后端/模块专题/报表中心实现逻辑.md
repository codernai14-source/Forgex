# 报表中心实现逻辑

> 分类：后端 / 模块专题  
> 版本：**V0.6.0**  
> 更新时间：**2026-04-17**

本文档说明报表中心分类、数据源、模板和双引擎集成在后端中的实现方式。

## 核心控制器

| 控制器 | 作用 |
|---|---|
| `ReportCategoryController` | 分类管理 |
| `ReportDatasourceController` | 数据源管理与测试 |
| `ReportTemplateController` | 模板管理 |

## 核心链路

```text
模板维护
  -> 分类归属
  -> 数据源绑定
  -> 模板内容保存
  -> 设计器 URL / 预览 URL 生成
```

## 双引擎

当前支持 `UReport2` 与 `JimuReport`，前端按模板或配置选择不同设计器与预览地址。
