# 基础数据与物料使用方式

> 分类：后端 / 模块专题  
> 版本：**V0.6.0**  
> 更新时间：**2026-04-17**

本文档说明基础数据与物料模块的使用与阅读方式。

## 适用场景

- 开发物料管理
- 开发供应商主数据
- 接入自动编码
- 排查基础模块启用状态

## 推荐做法

1. 物料编码统一走编码规则。
2. 状态、类型优先走字典。
3. 批量导入导出优先复用公共能力。

## 供应商主数据

供应商主数据前端入口：

```text
Forgex_MOM/Forgex_Fronted/src/views/basic/supplier/index.vue
```

页面通过 `FxDynamicTable` 承载列表和列配置，新增 / 编辑 / 详情共用 `BaseFormDialog`。当前供应商新增弹窗在调用处显式启用 `:mask-closable="true"`，因此打开新增供应商后点击弹窗外部遮罩会关闭弹窗。

使用建议：

1. 新增供应商时可启用自动生成编码，保存时由后端编码规则生成供应商编码。
2. 合作状态、信用等级、风险等级、供应商分级、审查状态等下拉项优先走数据字典。
3. 联系人和资质作为供应商主数据的子表信息随主表一起维护。
4. 导入、导出、模板下载优先复用公共导入导出能力。
