# 工作流

> 分类：后端 / 模块专题  
> 版本：**V0.5.0**  
> 更新时间：**2026-04-15**

本专题聚焦 `Forgex_Workflow` 当前已落地的审批配置、审批执行和前端页面入口。

## 一、模块定位

`Forgex_Workflow` 负责审批任务配置、审批发起、审批执行、追踪、补偿和回调。

当前服务名：`forgex-workflow`  
默认端口：`9005`

## 二、后端入口

### 2.1 控制器

| 控制器 | 说明 |
|---|---|
| `WfTaskConfigController` | 审批任务配置、草稿编辑、流程图、发布 |
| `WfExecutionController` | 发起审批、审批处理、委托、批量操作、补偿、统计 |
| `WfCallbackController` | 审批回调与外部联动 |

### 2.2 核心接口分组

| 能力 | 典型路径前缀 |
|---|---|
| 任务配置 | `/wf/task/config/*` |
| 审批执行 | `/wf/execution/*` |
| 回调通知 | `/wf/callback/*` |

## 三、前端入口

### 3.1 页面目录

| 页面 | 路径 | 说明 |
|---|---|---|
| 工作台首页 | `src/views/workflow/dashboard/index.vue` | 待办、昨日处理、抄送与统计 |
| 流程发起 | `src/views/workflow/execution/start.vue` | 发起审批 |
| 流程表单 | `src/views/workflow/execution/startForm.vue` | 发起表单页 |
| 低代码表单渲染 | `src/views/workflow/execution/components/LowCodeFormRenderer.vue` | 运行期表单渲染 |
| 我的待办/已办/已发起 | `src/views/workflow/myTask/*` | 审批处理与流程追踪 |
| 任务配置 | `src/views/workflow/taskConfig/*` | 审批配置、节点编辑、低代码设计器 |
| 补偿治理 | `src/views/workflow/governance/compensation/index.vue` | 异常补偿与超时重试 |

### 3.2 API 封装

当前前端工作流 API 主要分为：

- `src/api/workflow/taskConfig.ts`
- `src/api/workflow/execution.ts`

已覆盖：

- 任务配置分页、详情、保存、发布
- 发起审批、审批、驳回、转交、加签
- 批量审批、批量转交、批量提醒
- 我的待办、已办、已发起、抄送
- 审批委托、补偿、超时重试、仪表盘统计

## 四、当前能力清单

| 能力 | 当前结论 |
|---|---|
| 流程配置 | 已支持任务配置、草稿编辑、流程图维护 |
| 低代码表单 | 已支持设计器与运行期渲染器 |
| 审批执行 | 已支持发起、通过、驳回、取消 |
| 高级动作 | 已支持转交、加签、委托 |
| 批量处理 | 已支持批量审批、批量转交、批量提醒 |
| 追踪与治理 | 已支持动作日志、流程追踪、补偿治理、超时重试 |
| 工作台统计 | 已支持首页摘要与分析接口 |
