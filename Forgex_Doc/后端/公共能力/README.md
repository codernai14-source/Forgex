## 四、与业务模块的关系

| 公共能力 | 主要被谁复用 |
|---|---|
| `R<T>` / 状态码 / 异常 | 所有 Controller / Service |
| 国际化消息链路 | `Auth`、`Sys`、`Workflow` 等所有服务 |
| 消息模板 | `Sys`、`Workflow`、其他业务模块 |
| 自动填充注解 | 需要审计或展示创建/更新用户的模块 |
