# 编码规则、授权与初始化

> 分类：后端 / 配置与审计  
> 版本：**V0.5.0**  
> 更新时间：**2026-04-15**

本专题用于收敛当前已在 `Forgex_Sys` 中落地、但原正式导航中入口不清晰的配置类能力：

- 编码规则
- 软件授权
- 初始化向导
- 模块管理与在线能力相关配置入口

| `InitController` | 系统初始化状态与初始化执行 |

| `SysConfigController` | 系统基础配置 |

## 三、能力说明

### 3.1 编码规则

已形成配置驱动的编码规则能力，当前可见入口包括：

- 规则分页查询
- 规则保存与删除
- 规则测试
- 按 `ruleCode` 生成业务编码

建议把它作为 `Forgex_Basic`、`Forgex_Sys` 等业务模块复用的公共配置能力。

### 3.2 软件授权

当前授权能力已经进入运行链路：

- `LicenseController` 提供授权状态、申请信息、历史记录、刷新
- 网关过滤器会在鉴权前检查授权状态
- `GatewayAuthGlobalFilter` 已处理 `UNLICENSED / EXPIRED / GRACE / INVALID` 等场景

### 3.3 初始化向导

当前初始化链路已对外暴露：

- `GET /sys/init/status`
- `POST /sys/init/apply`

并在网关白名单中放行，适合作为安装后首次启动或演示环境初始化入口。

- 当前编码规则前端类型定义中存在历史字段命名痕迹，文档以实际页面/API 入口为准。
