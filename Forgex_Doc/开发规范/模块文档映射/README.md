# 模块文档映射

> 版本：**预览版 V0.5.0**  
> 更新时间：**2026-04-13**

本页用于说明原 `04-功能模块` 主题，如何映射到 `Forgex_Doc` 的正式阅读入口中。

## 一、映射规则

1. **`Forgex_Doc` 是正式阅读入口**，负责按主题归类和导航。
2. **`Forgex_Doc` 内部分类页与镜像页作为可提交入口**。
3. **一个文档只设置一个主归属分类**，但可以在其他分类页做“关联阅读”。
4. **所有分类页都应支持直接跳到原文**，形成可阅读的连续文档集。

## 二、分类入口

- 后端模块专题（见 `Forgex_Doc/后端/模块专题`）
- 后端公共能力（见 `Forgex_Doc/后端/公共能力`）
- 前端组件与页面（见 `Forgex_Doc/前端/组件与页面`）
- 数据库安全与配置（见 `Forgex_Doc/数据库/安全与配置`）
