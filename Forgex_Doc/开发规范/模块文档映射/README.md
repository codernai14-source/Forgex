## 一、映射规则

1. **`Forgex_Doc` 是正式阅读入口**，负责按主题归类和导航。
2. **`doc/开发规范/04-功能模块` 保留原文**，不直接搬迁大文件，避免破坏历史沉淀。
3. **一个文档只设置一个主归属分类**，但可以在其他分类页做“关联阅读”。
4. **所有分类页都应支持直接跳到原文**，形成可阅读的连续文档集。
